# Online Text Extractor (Streamlit)

A simple, offline-capable text extractor using Streamlit and Tesseract OCR.

## Prerequisites
1. **Python 3.9+**
2. **Tesseract-OCR Binary**: Must be installed on your system.
   - [Download for Windows](https://github.com/UB-Mannheim/tesseract/wiki)

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running
Run the application:
```bash
streamlit run app.py
```
